# PySpark ETL Pipeline - Smartphone Addiction Analysis

## Overview
This project demonstrates a real-world ETL pipeline built using PySpark on a smartphone usage dataset from kaggle.

## Architecture
- Bronze Layer: Raw data ingestion
- Silver Layer: Data cleaning and transformation
- Gold Layer: Aggregation and analysis

## Technologies
- PySpark
- Python

## Pipeline Steps
1. Ingest CSV data
2. Select relevant columns
3. Handle missing values
4. Perform feature engineering
5. Aggregate data for insights

## Key Insights
- Average screen time by age
- Addiction score by gender
- Academic impact analysis

## Learnings
- Built distributed data pipeline using PySpark
- Understood partitioning and shuffle
- Applied real-world ETL architecture

## Future Improvements
- Use Spark write APIs instead of pandas
- Optimize partitioning
